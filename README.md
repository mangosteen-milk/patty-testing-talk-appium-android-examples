# patty-testing-talk-appium-android-examples
本项目是YouTube频道 [@Patty-Testing-Talk](https://www.youtube.com/@Patty-Testing-Talk) 的Appium安卓自动化测试系列视频的配套代码和资源。

### 注意：项目中的代码和资源仅作学习用途，不能用于其他违反政策或不正当的用途

- 播放列表：https://www.youtube.com/@Patty-Testing-Talk/playlists

备注：AndroidDemo App做了一些改动，与前几期视频有些不一样的地方，主要是在首页增加了输入网址的地方，方便学习混合App(webview)自动化相关的内容
